# Proje Yol Haritası

## Aşama 1: Statik MVP

- HTML, CSS ve vanilla JavaScript ile vitrin sayfaları.
- Admin panelde ürün, kategori, sipariş ve ayar ekranları.
- Ürün fotoğraflarını dosyadan ekleme ve `localStorage` içinde saklama.
- Sepet, ödeme formu ve demo sipariş oluşturma.
- Netlify üzerinde statik yayın.

## Aşama 2: Gerçek Sipariş Takibi

- Backend API veya servis entegrasyonu.
- Ortak veritabanı ile admin panelden tüm müşteri siparişlerini görme.
- E-posta bildirimi ve sipariş durum bildirimleri.
- Admin giriş/şifre koruması.

## Aşama 3: Satılabilir Paket

- Firma renkleri, logo, kategori ve ürün örnekleri için kurulum dokümanı.
- Hazır tema varyasyonları.
- Netlify domain, form ve deploy dokümantasyonu.
- Bakım ve destek paketi.
