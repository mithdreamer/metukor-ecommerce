# Ecommerce Template

Küçük ölçekli firmalara uyarlanabilecek statik e-ticaret şablonu.

## Teknoloji

- HTML
- CSS
- Vanilla JavaScript
- Netlify statik yayın

## Çalıştırma

VS Code içinde Live Server ile `index.html` dosyasını açabilir veya terminalden statik sunucu başlatabilirsiniz:

```powershell
python -m http.server 5173
```

Sonra `http://localhost:5173` adresini açın.

## Önemli Not

Bu sürüm bir MVP ve demo altyapısıdır. Admin panel verileri `localStorage` üzerinde saklanır. Gerçek müşterilerden gelen siparişleri ortak admin panelde takip etmek için backend ve veritabanı gereklidir.

## Admin Giriş

Varsayılan kullanıcı adı `admin`, varsayılan şifre `123456`. Bu bilgiler admin ayarları ekranından değiştirilebilir.
